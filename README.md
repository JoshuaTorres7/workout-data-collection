# Fitness Activity Tracking API

## Project Overview
This project is a DIY API that collects and analyzes small fitness activity data.  
The goal of the project is to track workout habits and identify patterns such as workout frequency, duration, and intensity over time.

The API allows users to submit workout entries and retrieve stored data for analysis.


## Data Collected
Each workout entry contains the following fields:

- id
- workout_type
- duration_minutes
- intensity
- workout_date

Example workout types include:
- Cardio
- Strength
- Yoga
- HIIT


## Dataset


This dataset contains 50 workout entries used for testing and analyzing workout trends.
